# Projeto-de-An-lise-de-Dados-em-E-commerce-EDA-Feature-Engineering-e-Insights-de-Neg-cio
Projeto de análise exploratória e estratégica de dados de e-commerce utilizando Python, Pandas e visualização de dados. O estudo transforma dados transacionais reais em insights de negócio sobre receita, sazonalidade, cancelamentos, produtos e comportamento de clientes.
